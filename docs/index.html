คู่มือการใช้งานระบบ POS Desktop

ยินดีต้อนรับสู่ศูนย์รวมข้อมูลการใช้งาน และการตั้งค่าระบบอย่างมืออาชีพ

เริ่มต้นใช้งาน ตั้งค่า DB

การใช้งานเบื้องต้น

จัดการผู้ใช้งาน

การเพิ่ม/ลบ พนักงาน และการตั้งค่าสิทธิ์การใช้งานแบบละเอียด (Granular Permissions)

ระบบการขาย (POS)

ขั้นตอนการเปิดบิล การสแกนบาร์โค้ด และการรับชำระเงินผ่านช่องทางต่างๆ

คลังสินค้า

การจัดการสต็อกสินค้า การนำเข้าข้อมูลผ่าน Excel และการปรับปรุงจำนวนสินค้า

การเชื่อมต่อฐานข้อมูล

Standalone

ใช้งานเครื่องเดียว ดึงข้อมูลจากฐานข้อมูลในเครื่องตัวเอง (127.0.0.1)

Client

ใช้งานหลายเครื่อง ดึงข้อมูลจากเครื่องแม่ (Master Server) ผ่าน IP Network

Trial

โหมดทดลองใช้งาน ใช้ฐานข้อมูลแยกต่างหากเพื่อการทดสอบ

คำแนะนำ: ใช้ปุ่ม "Test Connection" ในหน้าตั้งค่าก่อนกดบันทึกเสมอ

การแก้ปัญหาเบื้องต้น

เชื่อมต่อฐานข้อมูลไม่ได้?

1. ตรวจสอบว่าเปิด Laragon หรือ MySQL Service หรือยัง?

2. ตรวจสอบ IP และ Port ว่าถูกต้องตามที่ตั้งค่าไว้ในหน้า DB Config หรือไม่

3. ตรวจสอบ Firewall ว่าไม่ได้บล็อกการสื่อสารของ Port 3306

ต้องการล้างข้อมูลยอดขายเก่าเพื่อประหยัดพื้นที่?

ไปที่เมนู ตั้งค่า > พื้นที่อันตราย เลือก "ลบข้อมูลเก่าตามช่วงเวลา" และเลือกวันที่ต้องการ

ลืมรหัสผ่าน Admin?

สามารถกดปุ่ม "Factory Reset" ในหน้าตั้งค่าระบบ (คำเตือน: ข้อมูลทั้งหมดจะหายไป) หรือติดต่อผู้พัฒนา
