# S.Mart POS - ESP32 Fingerprint Reader Hardware Guide
============================================================
เอกสารโครงสร้างฮาร์ดแวร์ การต่อสาย และการจัดวางบอร์ดระบบสแกนนิ้วอัจฉริยะ S.Mart POS
เวอร์ชัน 4.0 (อัปเดต: จอ LCD 2004 I2C + Active Buzzer GPIO32 + ปุ่มกด Single/Double Click)
============================================================

## 📌 สรุปพิกัดตำแหน่งขาต่อใช้งาน (Pinout & Wiring Diagram)

การเชื่อมต่อจะจ่ายไฟและพ่วงสายผ่าน **บอร์ดเขียวไข่ปลาอเนกประสงค์** โดยมีบอร์ด **ESP32 DevKit V1 (30-pin)** เป็นสมองกลหลัก เสียบอยู่บน **Female Pin Header** ที่บัดกรียึดอยู่กับบอร์ดเขียวอีกที เพื่อให้ถอดซ่อมบำรุงได้ง่าย

### 🔴 รางไฟหลักบนบอร์ดเขียว (Power Bus Rails)
* **ราง GND:** ต่อเชื่อมจากขา `GND` ของ ESP32 (ทุกอุปกรณ์ต่อขาลบร่วมกันที่นี่)
* **ราง 3.3V:** ต่อเชื่อมจากขา `3.3V (3U3)` ของ ESP32 (จ่ายไฟเลี้ยงระบบสัมผัสของ R307)
* **ราง 5V (VIN):** ต่อเชื่อมจากขา `VIN` ของ ESP32 (จ่ายไฟเลี้ยงชิป R307, จอ LCD, และ Buzzer)

> [!NOTE]
> **บันทึกสีสายไฟหลักที่คุณพ่อติใช้เชื่อมต่อใต้บอร์ด:**
> * 🟤 **สายไฟสีน้ำตาล** ➡️ ต่อเข้ากับ **ราง GND**
> * 🔵 **สายไฟสีน้ำเงิน** ➡️ ต่อเข้ากับ **ราง 3.3V**
> * 🔴 **สายไฟสีแดง**   ➡️ ต่อเข้ากับ **ราง VIN (5V)**

### 🔵 ตารางเชื่อมต่ออุปกรณ์เข้ากับขา ESP32

| อุปกรณ์ | ขาของอุปกรณ์ | ต่อเข้าจุดไหนของระบบ | สีสายไฟ / หมายเหตุ |
| :--- | :--- | :--- | :--- |
| **1. เครื่องสแกน R307S** | **VCC** | ➡️ รางไฟ **5V (VIN)** บนบอร์ดเขียว | *สายสีแดง (ไฟเลี้ยงเซนเซอร์หลัก)* |
| | **GND** | ➡️ รางไฟ **GND** บนบอร์ดเขียว | *สายสีดำ* |
| | **Touch 3.3V** | ➡️ รางไฟ **3.3V** บนบอร์ดเขียว | *สายสีขาว (วงจรสัมผัสสัมผัส)* |
| | **Touch Pin** | ➡️ ช่อง **GPIO 4 (D4)** ของ ESP32 | *สายสีน้ำเงิน (HIGH เมื่อมีนิ้วแตะ)* |
| | **TXD** | ➡️ ช่อง **GPIO 16 (RๅๅๅๅๅๅๅๅๅๅๅๅๅๅๅX2)** ของ ESP32 | *สายสีเหลือง (รับส่งข้อมูล)* |
| | **RXD** | ➡️ ช่อง **GPIO 17 (TX2)** ของ ESP32 | *สายสีเขียว (รับส่งข้อมูล)* |
| **2. จอ LCD 2004 + I2C**| **VCC** | ➡️ รางไฟ **5V (VIN)** บนบอร์ดเขียว | *สายสีม่วง (ไฟเลี้ยงจอ)* |
| | **GND** | ➡️ รางไฟ **GND** บนบอร์ดเขียว | *สายสีขาว (GND)* |
| | **SDA** | ➡️ ช่อง **GPIO 21 (D21/SDA)** ของ ESP32 | *สายสีน้ำเงิน (ข้อมูล I2C)* |
| | **SCL** | ➡️ ช่อง **GPIO 23 (D23)** ของ ESP32 | *สายสีเขียว (Clock I2C)* ⚠️ เปลี่ยนจาก D22 |

| **3. Active Buzzer** | **VCC / +** | ➡️ รางไฟ **3.3V** บนบอร์ดเขียว | *โมดูลแบบ 3 ขา (ย้ายมา 3.3V เพื่อแก้ลั่น)* |
| | **GND / -** | ➡️ รางไฟ **GND** บนบอร์ดเขียว | |
| | **I/O / S** | ➡️ ช่อง **GPIO 33 (D33)** ของ ESP32 | *ส่งสัญญาณควบคุมเสียง* ⚠️ เปลี่ยนจาก D32 หนีขาช็อต |
| **4. ปุ่มกด** | **VCC** | ➡️ รางไฟ **3.3V** บนบอร์ดเขียว | *สายดำ (W14) (ย้ายมา 3.3V เซฟบอร์ด)* |
| *(โมดูล 3 ขา)* | **OUT** | ➡️ ช่อง **GPIO 18 (D18)** ของ ESP32 | *สายขาว (X16)* ⚠️ ย้ายจาก D34 เพื่อให้ใช้ PULLUP/PULLDOWN ภายในได้ |
| | **GND** | ➡️ รางไฟ **GND** บนบอร์ดเขียว | *สายเทา (W18)* |

> [!IMPORTANT]
> **ฟังก์ชันปุ่มกด (GPIO 18):**
> * กด **1 ครั้ง** = เข้าโหมดออกพักชั่วคราว (ต้องสแกนนิ้วยืนยัน)
> * กด **2 ครั้งติดกัน** (< 500ms) = ส่งคำขอลงทะเบียนลายนิ้วมือไปยัง POS (`ENROLL_REQUEST`)

## 🛠️ ขั้นตอนการประกอบลงแผงจริง (เมื่อของส่งถึงมือ)

1. **ตัดตัวเมีย (Female Pin Header):**
   * เอาหัวตัวเมีย 17 ขา (1x17P) ที่สั่งซื้อมามาทาบกับขา ESP32
   * ใช้คีมตัดหรือเลื่อยแบ่งให้ได้ความยาวเท่ากับจำนวนขาของ ESP32 ข้างละ 15 ขา (หรือเท่ากับจำนวนขาจริงของบอร์ดพี่ติ) จำนวน 2 แถว
2. **บัดกรีเต้ารับตัวเมียลงบอร์ดเขียว:**
   * วางเต้ารับตัวเมียทั้ง 2 แถวลงบนตำแหน่งที่ต้องการบนบอร์ดเขียวไข่ปลา (เว้นช่องตรงกลางให้กว้างพอดีกับตัวถัง ESP32)
   * พลิกใต้บอร์ด บัดกรีขาล็อคเต้ารับทุกขาเข้ากับบอร์ดเขียวเพื่อความแข็งแรงสูงสุด
3. **ลากตะกั่วสะพานไฟใต้บอร์ด:**
   * ใช้เศษขาลวดทองแดงวางพาดและบัดกรีทำ **"ราง GND"**, **"ราง 3.3V"**, และ **"ราง 5V"** ใต้บอร์ดเขียว
   * ต่อสายไฟจากรูเต้ารับที่ตรงกับขา `GND`, `3.3V`, `VIN` ของ ESP32 ไปเชื่อมเข้ากับรางไฟที่เราสร้างไว้
4. **เชื่อมต่อพอร์ตข้างนอก:**
   * นำสายไฟจากเซนเซอร์ R307, จอ LCD, Buzzer และปุ่มกด มาบัดกรีลงรูว่างข้างๆ เต้ารับตัวเมีย จากนั้นลากตะกั่วเชื่อมต่อสายเหล่านี้เข้าไปหาขาใช้งานตามตารางด้านบน
5. **ประกอบร่างสำเร็จ:**
   * เสียบตัวบอร์ด ESP32 ลงไปบนเต้ารับตัวเมีย 
   * ยึดบอร์ดเขียวไข่ปลาเข้ากับ **ขาตั้งบอร์ดตู้เกมสีเหลือง** ที่ซื้อมา
   * ติดตั้งลงกล่องพักสายไฟสีขาว เจาะรูนำสายไฟออกและเชื่อมต่อสวิตช์หน้ากล่องให้เรียบร้อย พร้อมหยอดกาวร้อน/ซิลิโคนซีลรูนำสายไฟออก

## 🗺️ แผนผังการจัดวางชิ้นส่วนบนแผ่นบอร์ดเขียว 5x7 ซม. (Perfboard 5x7 cm Layout)

### 🔌 แนวทางการเดินลวดทองแดงสะพานไฟใต้บอร์ด (Under-Board Routing Rules)

1. **ราง GND หลัก (แนวนอน แถว A):**
   * ลากลวดทองแดงเปลือยยาวขนานขอบบอร์ดบนแถว A 
   * โยงขาสายจากเต้ารับตัวเมียรู **`E4` (GND)** และ **`N4` (GND)** ขึ้นไปเชื่อมที่แถว A นี้
2. **ราง 3.3V (แนวนอน แถว Q):**
   * ลากลวดทองแดงเปลือยพาดสั้นแถว Q จากรู 1 ถึง 5 
   * โยงขาสายจากเต้ารับตัวเมียรู **`N2` (3V3)** ลงมาเชื่อมที่แถว Q นี้
3. **ราง 5V/VIN (แนวนอน แถว S):**
   * ลากลวดทองแดงเปลือยพาดแถว S จากรู 1 ถึง 7
   * โยงขาสายจากเต้ารับตัวเมียรู **`E5` (5V)** และ **`N5` (VIN)** ลงมาเชื่อมที่แถว S นี้
4. **การต่อจุดรวมพอร์ต (Port Header):**
   * บริเวณแถว U, V, W, X ให้บัดกรี **Pin Header ตัวผู้ (Male Pin Header)** หรือช่องขั้วต่อ
   * ลากสายไฟสะพานทองแดงหุ้มฉนวน (สีชมพู/สีเขียว) จากจุดต่อเหล่านี้ วิ่งลอดพาดข้ามใต้บอร์ดไปบัดกรีเชื่อมเข้าที่เต้ารับตัวเมียของ ESP32 ตามสเปก Pinout เพื่อความเป็นระเบียบและไม่รกรุงรังด้านหน้าบอร์ดครับ
   
   > [!NOTE]
   > **พิกัดพอร์ตบัดกรีสายเครื่องสแกน R307 บนบอร์ดไข่ปลา แถว X (พิกัดรู X1 ถึง X11):**
   * 🔴 รู **`X1`** ➡️ ต่อสาย **สีแดง (VCC)**
   * 🟤 รู **`X3`** ➡️ ต่อสาย **สีดำ (GND)**
   * 🟡 รู **`X5`** ➡️ ต่อสาย **สีเหลือง (TXD)**
   * 🟢 รู **`X7`** ➡️ ต่อสาย **สีเขียว (RXD)**
   * 🔵 รู **`X9`** ➡️ ต่อสาย **สีน้ำเงิน (Touch Pin)**
   * ⚪ รู **`X11`** ➡️ ต่อสาย **สีขาว (Touch 3.3V)**

   > [!NOTE]
   > **พิกัดพอร์ตบัดกรีปุ่มกดเปลี่ยนโหมด (3 ขา) เพื่อเว้นระยะปลอดภัย (ข้ามช่องกันช็อต):**
   * ⚫ รู **`W14`** ➡️ ต่อสาย **สีดำ (VCC)** ➡️ ลากสายใต้บอร์ดไปราง **3.3V**
   * ⚪ รู **`X16`** ➡️ ต่อสาย **สีขาว (OUT/Signal)** ➡️ โยงสายใต้บอร์ดเข้าขา **GPIO 18 (D18)**
   * 🔘 รู **`W18`** ➡️ ต่อสาย **สีเทา (GND)** ➡️ ลากสายใต้บอร์ดเข้าแถว **GND**
