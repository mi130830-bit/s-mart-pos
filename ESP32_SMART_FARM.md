# ESP32_SMART_FARM.md: Lazy Farm Project (แผนฟาร์มผักคนขี้เกียจ)

โปรเจกต์ของเล่นกึ่งใช้งานจริงสำหรับ **พี่ติ** เพื่อรดน้ำผักอัตโนมัติโดยใช้บอร์ด **ESP32** ร่วมกับเซนเซอร์ความชื้นในดินและระบบควบคุมปั๊มน้ำ ผ่านหน้าเว็บอนิเมชันพิกเซลอาร์ตสุดน่ารัก

---

## 🛠️ รายการอุปกรณ์ (Hardware List)
1. **ESP32 DevKit v1** (สมองหลัก + Wi-Fi Web Server)
2. **Capacitive Soil Moisture Sensor v1.2** (เซนเซอร์วัดความชื้นในดินแบบต้านทานการกัดกร่อน)
3. **Relay Module 5V 1-Channel** (สวิตช์ตัดต่อไฟคุมปั๊มน้ำ)
4. **Mini Submersible Water Pump 5V** (ปั๊มน้ำจิ๋ว 5V) + สายยางซิลิโคน
5. **Water Level Sensor** (เซนเซอร์วัดระดับน้ำป้องการปั๊มไหม้เวลาแห้ง)

---

## 🔌 การต่อวงจร (Wiring Diagram)
* **Soil Sensor Analog Out** -> **ESP32 GPIO 36 (VP)**
* **Relay IN** -> **ESP32 GPIO 23**
* **Water Level Sensor Out** -> **ESP32 GPIO 34**
* **VCC/GND** ของทุกอุปกรณ์เชื่อมต่อแหล่งจ่ายไฟ 5V (GND ร่วมกับ ESP32)

---

## 🎨 การออกแบบหน้าเว็บ (Pixel Art Web UI - Stardew Valley Style)
หน้าเว็บ HTML/CSS/JS แบบ Single Page ที่รันบน ESP32 และดึงข้อมูลแบบ Real-time:

* **Happy Mode (ชื้นดี > 60%):** ผักพิกเซลสไตล์เรโทร 8-bit เต้นดุ๊กดิ๊กบนผืนดิน มีไอคอนพระอาทิตย์ยิ้มแย้ม
* **Thirsty Mode (ดินแห้ง < 40%):** ผักทำหน้าเศร้าคอตก มีไอคอนหยดน้ำสีแดงกระพริบเตือน
* **Watering Mode (กำลังรดน้ำ):** มีอนิเมชันฝนพิกเซลตกลงมาใส่ผัก พร้อมแถบเกจวัดความชื้นค่อยๆ สูงขึ้น

---

## 🔒 กฎเหล็กความปลอดภัยของระบบ (Safety Guard Logic)
1. **Water Time Limit:** สั่งปั๊มทำงานสูงสุดครั้งละไม่เกิน 10 วินาที เพื่อป้องกันปั๊มค้างแล้วน้ำท่วม
2. **Post-Water Cooldown:** รดเสร็จแล้วต้องรอคูลดาวน์ 1 ชั่วโมง ค่อยประเมินค่าความชื้นใหม่
3. **Dry Burn Protection:** หากระดับน้ำในถังต่ำเกินไป ให้ตัดไฟเลี้ยงปั๊มทันทีและส่ง Line Notify เตือนพนักงาน
